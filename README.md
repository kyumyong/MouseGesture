# Mouse Gesture Program

사용자가 마우스 오른쪽 버튼을 누르고 있는 동안의 마우스 동작을 인식하여, 해당 동작에 매핑된 기능을 실행하는 프로그램입니다.

## 기능

| 제스처 | 기능 |
| --- | --- |
| ↙ (오른쪽 위에서 왼쪽 아래로) | 활성 창 최소화 | (gemini는 ↘ 기호를 쓰는 오류 발생)
| ↗ (왼쪽 아래에서 오른쪽 위로) | 활성 창 최대화/이전 크기로 복원 |
| → (왼쪽에서 오른쪽으로) | 다음 페이지/폴더로 이동 |
| ← (오른쪽에서 왼쪽으로) | 이전 페이지/폴더로 이동 |
| ↑ (아래에서 위로) | 복사 (Ctrl+C) |
| ↓ (위에서 아래로) | 붙여넣기 (Ctrl+V) |
| ↖ (오른쪽 아래에서 왼쪽 위로) | 활성 창 닫기 (Alt+F4) |

## 설치

필요한 라이브러리를 설치합니다.

```bash
pip install pynput pyautogui pygetwindow
```

## 실행

프로그램을 실행합니다.

```bash
python main.py
```
